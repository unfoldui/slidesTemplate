ES6 Promises

Promises give us a way to handle asynchronous processing in a more synchronous fashion.

ES6 Promises

Implementation

export const myFirstPromise = new Promise((resolve, reject) => {
  if (condition === true)
    resolve(someValue)
  else {
    reject("failure reason")
  }
});

Usage

import {myFirstPromise} from '../API'

   myFirstPromise()
     .then(res => console.log(res))
     .catch(error => console.log(error))

Fetch API

Fetch API

  • Simpler and cleaner API than XMLHttpRequest
  • Uses ES6 Promises instead of callbacks

Using Fetch API

GET Request


fetch('http://example.com/movies.json')
  .then(res => res.json())
  .catch(console.error)

Using Fetch API

POST Request


fetch(url, {
  method : "POST",
  body   : JSON.stringify(data),
  headers: new Headers({
    "Content-Type": "application/json"
  })
})
.then(res => res.json())
.catch(console.error)
